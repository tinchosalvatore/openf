# openf

A simple CLI tool to open any file from your terminal using the system's default application.

## Installation

Run the installer script:
```bash
python3 install.py
```

## Usage

Simply type `openf` followed by the file you want to open:
```bash
openf my_document.pdf
openf image.png
```

## Uninstallation

Run the uninstaller script:
```bash
python3 uninstall.py
```
